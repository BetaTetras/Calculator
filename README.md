## ⚙️ | Fonctionnement et déploiement 
### Requis :
- N'importe qu'elle PC sous Windows (Tester sur Windows 11)
### Lancement :
Utiliser l'executable : `C:\[...]\Calculator\Calculatrice\Calculatrice\bin\Debug\net8.0-windows\Calculatrice.exe`
